# 改修要件定義 - endless-runner v1.1

## 概要

既存のエンドレスランナー（Phaser.js + TypeScript）に対する改修。
ライフシステム・新障害物・アイテムを追加し、ゲームとしての深みを増す。

---

## 1. ライフシステム

### FR-01: ライフ上限の設定
- プレイヤーのライフは最大 **3** とする
- ゲーム開始時にライフ = 3 で始まる

### FR-02: ダメージ処理
- 障害物に衝突した場合、ライフを **1 減らす**
- ライフが 0 になった時点でゲームオーバー
- ダメージ後に **無敵時間（約 1.5 秒）** を設ける（連続ダメージ防止）

### FR-03: ライフ表示
- HUD にライフを常時表示する（例: `♥ ♥ ♥` / `LIFE: 3`）

---

## 2. 直線飛来障害物（ProjectileObstacle）

### FR-04: 3種類の高さを持つ障害物
水平に直線飛来する障害物を追加。高さのレーンは以下の3種類：

| レーン | 高さ（画面Y座標） | プレイヤーとの関係 |
|--------|-------------------|-------------------|
| Ground | ≈ 375 | 地上走行中に当たる |
| Mid    | ≈ 240 | 1段ジャンプ頂点付近で当たる |
| High   | ≈ 120 | 2段ジャンプ頂点付近で当たる |

### FR-05: 飛来挙動
- 右端（画面外）からスポーン、一定速度で左へ直進
- 速度はゲーム速度（`gameSpeed`）に連動
- 画面外左端でデアクティベート

### FR-06: スポーン条件
- 難易度レベル2以上からスポーン開始
- レーンはランダムに選択（Ground / Mid / High を均等確率）
- 既存の障害物スポーンとは独立したタイマーで管理

---

## 3. 得点アイテム（ScoreItem）

### FR-07: 得点アイテムの出現
- 3つのレーン（Ground / Mid / High）に出現
- 出現レーンはランダム選択
- 収集すると **スコアボーナス +100** 加算

### FR-08: 得点アイテムの視認性
- 色や形で障害物・回復アイテムと区別できること（例：金色の星型）

---

## 4. 回復アイテム（RecoveryItem）

### FR-09: 回復アイテムの出現
- 3つのレーン（Ground / Mid / High）に出現
- 出現レーンはランダム選択
- 収集すると **ライフを 1 回復**（上限: MAX_LIVES = 3）

### FR-10: 回復アイテムの視認性
- 色や形で障害物・得点アイテムと区別できること（例：緑色のハート型）

### FR-11: 出現頻度
- スコアアイテム・回復アイテムともに既存のパワーアップアイテムとは独立したマネージャーで管理
- スポーン間隔: 約 6 秒（スコアアイテム）、約 12 秒（回復アイテム）

---

## 5. 無敵パワーアップアイテム（InvincibilityPowerUp）

### FR-12: 無敵アイテムの出現
- 3つのレーン（Ground / Mid / High）に出現
- 出現レーンはランダム選択
- 既存の PowerUp システム（PowerUp 抽象クラス・PowerUpManager）に従って実装

### FR-13: 無敵効果
- 収集すると **一定時間（5秒）無敵状態** になる
- 無敵中は障害物に衝突してもダメージを受けない
- 無敵時間は HUD に残り秒数を表示（例: `★ INVINCIBLE 4s`）
- 無敵中のプレイヤーを視覚的に区別できること（例: 白くフラッシュ）

### FR-14: DoubleJump との共存
- DoubleJump パワーアップと無敵パワーアップは同時に適用可能
- 各効果は独立してタイマー管理される

---

## 6. 非機能要件

### NFR-01: 技術スタック
- 既存の Phaser.js + TypeScript 構成を維持

### NFR-02: オブジェクトプール
- 新しいオブジェクト（ProjectileObstacle, ScoreItem, RecoveryItem）もプールパターンで実装

### NFR-03: テスト
- 既存テスト（13件）を壊さないこと
- 新機能に対する単体テストを追加

---

## 影響を受ける既存ファイル

| ファイル | 変更内容 |
|---------|---------|
| `GameConfig.ts` | MAX_LIVES, PROJECTILE_Y定数, アイテム定数を追加 |
| `ObstacleManager.ts` | ProjectileObstacle プールとスポーン処理を追加 |
| `GameScene.ts` | LifeManager, ItemManager を統合、衝突処理を更新 |
| `GameHUD.ts` | ライフ表示・無敵パワーアップ表示を追加 |
| `ScoreManager.ts` | addBonus() メソッドを追加 |

## 新規ファイル

| ファイル | 役割 |
|---------|------|
| `managers/LifeManager.ts` | ライフ管理（最大値・ダメージ・回復・無敵時間） |
| `objects/ProjectileObstacle.ts` | 直線飛来障害物 |
| `objects/ScoreItem.ts` | 得点アイテム |
| `objects/RecoveryItem.ts` | 回復アイテム |
| `managers/ItemManager.ts` | ScoreItem / RecoveryItem のスポーンと収集管理 |
| `powerups/InvincibilityPowerUp.ts` | 無敵パワーアップ効果の実装 |
