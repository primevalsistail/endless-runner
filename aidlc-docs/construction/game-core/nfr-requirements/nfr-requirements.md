# NFR Requirements — Unit 1: game-core

> **v2 更新**: Phaser.js（ブラウザゲーム）向けに更新

## パフォーマンス

| ID | 要件 | 基準値 |
|----|------|--------|
| PERF-01 | フレームレート | 60fps を安定維持（Phaser のデフォルト） |
| PERF-02 | ゲーム起動時間 | ブラウザ読み込みからプレイ可能まで 3 秒以内 |
| PERF-03 | メモリ使用量 | 通常プレイ中 100MB 以下 |

## 互換性

| ID | 要件 | 基準値 |
|----|------|--------|
| COMPAT-01 | 対応ブラウザ | Chrome・Safari・Firefox（最新版） |
| COMPAT-02 | iPhone Safari | iOS 16.0+ Safari で動作・横向き対応 |
| COMPAT-03 | 画面向き | Landscape を推奨（portrait でも崩れない） |
| COMPAT-04 | タッチ操作 | タップ・クリック両対応 |

## 信頼性

| ID | 要件 | 対応方針 |
|----|------|---------|
| REL-01 | オフライン動作 | 外部通信なし。ページ読み込み後はオフライン動作 |
| REL-02 | データ保護 | localStorage に保存。ブラウザキャッシュ削除まで保持 |
| REL-03 | 例外処理 | ゲームループ内の未処理例外をキャッチし安全にゲームオーバーへ遷移 |

## 保守性

| ID | 要件 | 方針 |
|----|------|------|
| MAINT-01 | 型安全 | TypeScript を使用し型エラーをコンパイル時に検出 |
| MAINT-02 | テスト | ScoreManager・DifficultyManager のビジネスロジックをユニットテスト |
| MAINT-03 | 拡張性 | Obstacle・PowerUp の抽象クラスで OCP 準拠。将来の拡張・ジャンル変更に備える |

## スコープ外

| 項目 | 理由 |
|------|------|
| セキュリティ | オフライン・個人データなし |
| スケーラビリティ | サーバーなし |
| オーディオ | v1 スコープ外 |
| アクセシビリティ | v1 スコープ外 |
| App Store 公開 | 技術デモ完成後に判断 |
