# Business Rules — Unit 1: game-core

## ジャンプルール

| # | ルール |
|---|--------|
| JR-01 | プレイヤーは `isOnGround = true` のときのみ通常ジャンプできる |
| JR-02 | 長押しの追加力は `MAX_JUMP_HOLD_TIME` を超えて適用されない |
| JR-03 | 空中ジャンプは `DoubleJumpPowerUp` が有効 かつ `airJumpUsed = false` のときのみ実行可能 |
| JR-04 | 地面着地時に `airJumpUsed` を `false` にリセットする |
| JR-05 | カウントダウン中はジャンプ入力を受け付けない |

---

## スコアルール

| # | ルール |
|---|--------|
| SR-01 | スコアはゲームプレイ中のみ加算される（カウントダウン中・ゲームオーバー後は加算しない） |
| SR-02 | スコア = floor(distanceTraveled × SCORE_MULTIPLIER) |
| SR-03 | ハイスコアは `currentScore > storedHighScore` のときのみ更新・保存する |
| SR-04 | スコアはセッション開始時に 0 にリセットする |

---

## 難易度ルール

| # | ルール |
|---|--------|
| DR-01 | ゲーム速度は `MAX_SPEED` を超えない |
| DR-02 | 難易度レベルはセッション内で下がらない（単調増加） |
| DR-03 | レベルアップは走行距離が閾値を超えたフレームで即時適用する |
| DR-04 | Moving障害物はLevel 3以降からスポーンする |

---

## 障害物ルール

| # | ルール |
|---|--------|
| OR-01 | 障害物は画面右端 + margin の位置にスポーンする |
| OR-02 | 障害物の左端が画面左端を超えたフレームで削除する |
| OR-03 | カウントダウン中は障害物をスポーンしない |
| OR-04 | 隣接する障害物間に最低限のギャップを確保する（プレイヤーが通過できる幅） |

---

## パワーアップルール

| # | ルール |
|---|--------|
| PR-01 | 同種のパワーアップを取得した場合、残り時間をリセット（duration に戻す）して継続する |
| PR-02 | 異なる種類のパワーアップは同時に複数有効にできる |
| PR-03 | 残り時間が 0 になった時点でその種類の効果を即時解除する |
| PR-04 | カウントダウン中はパワーアップをスポーンしない |
| PR-05 | パワーアップアイテムのスポーン位置はプレイヤーがジャンプで到達できる高さ範囲内とする |

---

## 衝突ルール

| # | ルール |
|---|--------|
| CR-01 | プレイヤーの当たり判定と障害物の当たり判定が重なった瞬間にゲームオーバーとする |
| CR-02 | ゲームオーバー後はそれ以上の衝突判定を行わない |
| CR-03 | パワーアップアイテムとの衝突はゲームオーバーではなく効果適用とする |

---

## ゲーム開始ルール

| # | ルール |
|---|--------|
| GS-01 | ゲーム開始時は必ずカウントダウン（3・2・1・GO!）を表示する |
| GS-02 | カウントダウン中はゲームループの進行（スクロール・スポーン・スコア加算）を停止する |
| GS-03 | 「GO!」表示の直後にゲームループを開始する |
