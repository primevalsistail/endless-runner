# Integration Test Instructions: BGM & 効果音

## テストシナリオ一覧

### シナリオ 1: メニューBGM再生
**目的**: MenuScene表示でメニューBGMが自動再生されること

**手順**:
1. ゲームを開始（`npm run dev`）
2. MenuSceneが表示される
3. ブラウザのタブをクリック（Autoplay unlock）

**期待結果**: メニューBGMがループ再生される

---

### シナリオ 2: BGM切り替え（Menu → Game）
**目的**: ゲーム開始時にBGMがゲームBGMに切り替わること

**手順**:
1. MenuSceneでBGMが流れていることを確認
2. "TAP TO START" をクリック
3. GameSceneに遷移

**期待結果**:
- メニューBGMが停止
- ゲームBGMがループ開始

---

### シナリオ 3: ゲームオーバーBGM停止
**目的**: ゲームオーバー時にBGMが停止しgameover SFXが鳴ること

**手順**:
1. ゲームプレイ中（ゲームBGM流れている）
2. ライフを3つ失ってゲームオーバーにする

**期待結果**:
- ゲームBGMが停止
- gameover SFXが約100ms後に再生

---

### シナリオ 4: ポーズ音声制御
**目的**: ポーズ時にSFX停止・BGM音量ダウンが動作すること

**手順**:
1. ゲームプレイ中（BGM流れている）
2. ESCキー or ポーズボタンでポーズ
3. ポーズ中の音量を確認

**期待結果**:
- BGMが小さくなる（PAUSE_BGM_VOLUME = 0.15）
- ポーズ解除後に元の音量に戻る

---

### シナリオ 5: SFX 全種確認
**目的**: 全7種のSFXが正しいタイミングで再生されること

| SFX | トリガー | 確認方法 |
|-----|---------|---------|
| jump | 地上からジャンプ | タップ/クリック |
| double-jump | 空中でタップ（2段ジャンプパワーアップ後） | パワーアップ取得後に空中タップ |
| hit | 障害物に当たる | 障害物に衝突 |
| coin | コイン取得 | コインアイテムに触れる |
| heal | 回復アイテム取得 | バンデージに触れる |
| powerup | パワーアップ取得 | シールド/ブーツに触れる |
| gameover | ゲームオーバー | ライフ0で倒れる |

---

### シナリオ 6: 音量スライダーUI
**目的**: スライダーが正常に動作し設定が永続化されること

**手順**:
1. MenuScene でBGMスライダーをドラッグ
2. 音量変化を確認
3. ページリロード
4. スライダーが前回位置に戻ることを確認

**期待結果**: 設定がLocalStorageに保存・復元される

---

### シナリオ 7: Retryボタンでゲーム再開
**目的**: GameOverSceneからRetryしたとき、ゲームBGMが再開されること

**手順**:
1. ゲームオーバー → GameOverSceneへ遷移
2. "RETRY" ボタンをクリック

**期待結果**: GameScene のゲームBGMが再開される
